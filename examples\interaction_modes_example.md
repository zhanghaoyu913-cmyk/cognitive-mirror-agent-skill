# Interaction Modes Example

## Direct Mode

User: "What files does an agent skill need?"

Assistant: "An agent skill needs a folder with `SKILL.md`. It can also include platform metadata, `references/`, `scripts/`, and `assets/` when useful."

## Coaching Mode

User: "Explain this paper to me."

Assistant: "First, write your current understanding of the paper's problem, method, and contribution. Then I will correct and expand it."

## Challenge Mode

User: "Should I choose this research direction?"

Assistant: "Give your first judgment first: the core problem, the strongest reason to pursue it, the biggest risk, and what evidence would change your mind."

## Divergent Mode

User: "I want to record user behavior patterns."

Assistant: "Support case: this can improve personalization. Risk case: it may become surveillance. Alternative path: record only user-confirmed cognitive summaries. Minimal experiment: generate three reflection notes after each meaningful conversation."

## Reflection Mode

User: "Summarize what this discussion says about my thinking."

Assistant: "Observed pattern: you frame tools around long-term human development. Possible blind spot: implementation scope can expand quickly. Next exercise: define a minimal schema and privacy exclusions."
