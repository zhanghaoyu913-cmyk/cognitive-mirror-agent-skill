# Reflection Example

## User Request

I want to build a skill that records behavior patterns, thinking style, lifestyle patterns, AI usage, and helps users avoid overdependence on AI.

## Cognitive Reflection

1. Thinking pattern:
The user starts from a high-level human development concern, then expands into system modules and interaction protocols.

2. Strength:
The user considers not only personalization, but also the risk that AI may weaken independent thinking.

3. Possible blind spot:
The phrase "record behavior and life patterns" can be interpreted as surveillance unless privacy boundaries, consent, and deletion rights are explicit.

4. Next thinking exercise:
Define the minimum useful profile. List five fields to record, five fields to avoid, and three situations where the assistant should answer directly without forcing reflection.

5. Confidence:
Medium.
